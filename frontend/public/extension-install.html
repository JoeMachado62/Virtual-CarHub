💡 Quick Tip
The file will download to your Downloads folder. Keep this page open for the next steps!
2

Extract the Extension Folder

Find the downloaded file in your Downloads folder and extract it:

  • Windows: Right-click → "Extract All" → Choose Desktop
  • Mac: Double-click the .tar.gz file (it extracts automatically)

You should now see a folder called chrome-extension on your Desktop

3

Install in Chrome Browser

Follow these steps to load the extension:

A. Open Chrome Extensions Page

Copy and paste this into Chrome's address bar:

chrome://extensions/

Or click: Menu (⋮) → More Tools → Extensions

B. Enable Developer Mode

Find the toggle in the top-right corner and turn it ON

Developer mode ON

Toggle should be blue when enabled

C. Load the Extension

Click "Load unpacked" button (appears after enabling Developer mode)

Navigate to your Desktop and select the chrome-extension folder

D. Pin the Extension (Recommended)

Click the puzzle piece icon 🧩 in Chrome's toolbar

Find "Auto Fill Agent" and click the pin 📌 icon

✅ Success!
You should now see the Auto Fill Agent icon in your Chrome toolbar. Click it to login!

Frequently Asked Questions

❓ I don't see "Load unpacked" button
Make sure Developer mode is enabled (toggle in top-right corner). The button appears below the toggle.
❓ Extension shows an error after loading
Make sure you selected the correct folder (chrome-extension) and all files are present. Try refreshing the extension page.
❓ Can I delete the folder after installation?
No! Chrome needs the folder to run the extension. Keep it on your Desktop or move it to a permanent location (not Downloads).
❓ Will this work on other browsers?
Currently Chrome and Edge only. Firefox and Safari are not supported at this time.
❓ Is my data secure?
Yes! The extension only communicates with portal.pinnacleautofinance.com. No data is sent to third parties. All communication uses HTTPS encryption.
